# Transferencia-de-Calor-con-Python-y-Matlab
Repositorio oficial del libro académico "Aplicaciones de la Transferencia de Calor con Python® y MATLAB®", desarrollado en la Universidad Nacional de Colombia (2026).  Incluye: - Código en Python y MATLAB - Simulaciones numéricas - Modelos de transferencia de calory resolución computacional de sus ecuaciones más útiles y representativas.
